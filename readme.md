### Tech Stack ###

* Docker used to contain dependencies and tooling
* `node` >= 8.9.0
* `npm` >= 5.5.1
* `TypeScript` >= 3.0.1

### How to use(using docker)

```
clone repository
cd docker
chmod 777 dockerCompose.sh
./dockerCompose.sh

Now docker image is created, run your docker image by command below

docker-compose up
```

### Use without docker

```
clone
cd to project
npm i
npm start
```
